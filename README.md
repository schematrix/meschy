# Meschy

Meschy is an open-source LoRa + nRF52840 development board built by Schematrix. It is designed for Meshtastic-based communication, IoT prototypes, GPS tracking, remote sensor networks, and field-ready mesh networking projects.

## Features

* nRF52840 processor
* LoRa SX1262 radio module
* Meshtastic firmware compatibility
* BLE 5.0 and USB-C connectivity
* GPIO support for external sensors and modules
* Infrastructure-free mesh communication
* Suitable for disaster response, outdoor communication, agriculture, and IoT applications

## About

Meschy is not just a communication device. It is a flexible development platform for building independent, low-power, long-range wireless projects.
